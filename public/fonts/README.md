# Zhouli Serif Web Fonts

These web fonts are derived from Adobe Source Han Serif CN 2.003R and are
licensed under the SIL Open Font License 1.1. See `LICENSE.txt`.

- `zhouli-serif-ui-400/500/600.woff2`: small subsets for the fixed interface.
- `zhouli-serif-full-400/600.woff2`: complete Simplified Chinese coverage,
  loaded only for user-entered text, generated results, and card export.

Source release: https://github.com/adobe-fonts/source-han-serif/releases/tag/2.003R
